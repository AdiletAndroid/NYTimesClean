# NYTimesClean

This is small training MVP project with descrition and links to original articles of New York Times.

Requirements:
Android 5.0 Lollipop (API 21) and above.
Android Studio 4.0 and above.

Dependencies:
Retrofit 2.9.0
OkHttp 5.0.0-alpha.11
Room 2.5.1
Koin 3.4.0
Coil 2.3.0
Coroutines 1.6.4
Paging 3.1.1

Arch Pattern: MVP
